# HelloScalatra #
Before, Installing java, scala, sbt.
## Instralltion Scalatra ##
`$ sbt new scalatra/scalatra.g8`
## Build & Run ##

```sh
$ cd helloscalatra
$ sbt
> jetty:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.
